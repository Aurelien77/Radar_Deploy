Stimulus

Framework JavaScript minimaliste pour améliorer le HTML existant

Présentation

Stimulus est un framework JavaScript léger conçu pour améliorer le HTML existant. Contrairement aux frameworks SPA (Single Page Application), Stimulus ajoute de la réactivité en se concentrant sur le DOM déjà présent, parfait pour enrichir des pages server-rendered.

Principales caractéristiques

  • Ajout progressif de la réactivité sans réécrire le HTML existant.
  • Basé sur des "controllers" pour organiser le code JavaScript.
  • Pas de Virtual DOM, tout se passe directement sur le DOM réel.
  • Léger et facile à intégrer avec Rails, Django, Laravel ou n’importe quel backend.
  • Syntaxe simple et intuitive pour manipuler des éléments HTML.

Exemple d'utilisation


<div data-controller="hello">
  <button data-action="click->hello#greet">Say Hello</button>
</div>


import { Application } from "stimulus";
import HelloController from "./controllers/hello_controller";

const application = Application.start();
application.register("hello", HelloController);

// hello_controller.js
import { Controller } from "stimulus";

export default class extends Controller {
  greet() {
    alert("Hello, Stimulus!");
  }
}

Avantages

  • Très léger et facile à apprendre.
  • Permet de rendre le HTML interactif sans complexité SPA.
  • Compatible avec n’importe quel backend et architecture server-rendered.
  • Favorise une architecture simple et maintenable.
