jQuery

Bibliothèque JavaScript simplifiant la manipulation du DOM et les requêtes AJAX

Présentation

jQuery est une bibliothèque JavaScript légère et rapide, conçue pour simplifier la manipulation du DOM, la gestion des événements, les animations et les requêtes AJAX sur les sites web. Elle a été très populaire avant que les frameworks modernes comme React et Vue ne se généralisent.

Principales caractéristiques

  • Sélecteurs CSS simplifiés pour manipuler facilement les éléments du DOM.
  • Gestion cross-browser des événements et compatibilité avec tous les navigateurs.
  • Effets et animations simples à implémenter.
  • Requêtes AJAX simplifiées pour charger du contenu dynamique.
  • Extensible grâce à de nombreux plugins disponibles.

Exemple d'utilisation

// Modifier le texte d'un élément
$('#monDiv').text('Texte mis à jour');

// Cacher un élément au clic d'un bouton
$('#monBouton').click(function() {
    $('#monDiv').fadeOut();
});

Avantages

  • Simplifie grandement le code JavaScript natif.
  • Compatible avec la plupart des navigateurs.
  • Grand nombre de plugins et communauté active.
  • Idéal pour des projets légers ou existants avec beaucoup de code jQuery.
