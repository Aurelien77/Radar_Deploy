Preact

Bibliothèque JavaScript légère pour construire des interfaces utilisateur

Présentation

Preact est une bibliothèque JavaScript rapide et légère (≈3KB) pour construire des interfaces utilisateur. Elle propose une API très proche de React, ce qui facilite la migration ou l'utilisation conjointe.

Principales caractéristiques

  • Extrêmement légère et performante.
  • Compatible avec l'écosystème React (JSX, hooks, composants).
  • Support de la réactivité et des composants fonctionnels.
  • Idéal pour des projets nécessitant une faible taille de bundle.

Exemple d'utilisation

import { h, render, Component } from 'preact';

class App extends Component {
    render() {
        return <h1>Bonjour Preact !</h1>;
    }
}

render(<App />, document.body);

Avantages

  • Faible taille et excellente performance.
  • Syntaxe proche de React pour une adoption facile.
  • Support de JSX et des hooks modernes.
  • Large compatibilité avec les projets existants React.
