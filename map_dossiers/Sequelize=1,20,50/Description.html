Sequelize

ORM (Object-Relational Mapping) pour Node.js et bases SQL

Présentation

Sequelize est un ORM pour Node.js qui facilite l'interaction avec les bases de données relationnelles telles que MySQL, PostgreSQL, SQLite et MSSQL. Il permet de travailler avec des modèles JavaScript plutôt que d’écrire directement des requêtes SQL.

Principales caractéristiques

  • Support de MySQL, PostgreSQL, SQLite et MSSQL.
  • Définition de modèles et relations entre tables.
  • Migrations et synchronisation de base de données.
  • Requêtes SQL générées automatiquement.
  • Transactions et validations intégrées.

Exemple d'utilisation

const { Sequelize, DataTypes } = require('sequelize');
const sequelize = new Sequelize('sqlite::memory:');

const User = sequelize.define('User', {
  username: {
    type: DataTypes.STRING,
    allowNull: false
  },
  birthday: DataTypes.DATE
});

(async () => {
  await sequelize.sync();
  const jane = await User.create({ username: 'Jane', birthday: new Date(1990, 6, 20) });
  console.log(jane.toJSON());
})();

Avantages

  • Abstraction SQL pour un code JavaScript plus lisible.
  • Supporte les relations complexes et les transactions.
  • Migrations faciles à gérer pour la base de données.
  • Communauté active et documentation complète.
