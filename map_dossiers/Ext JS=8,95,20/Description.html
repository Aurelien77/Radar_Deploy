Ext JS

Framework JavaScript pour le développement d’applications web riches et d’entreprise

Présentation

Ext JS est un framework JavaScript commercial et open-source (en partie) développé par Sencha. Il permet de construire des applications web interactives, complexes et orientées entreprise, avec une interface riche et un grand nombre de composants intégrés.

Principales caractéristiques

  • Large bibliothèque de composants UI prêts à l’emploi (grilles, tableaux, arbres, menus, etc.).
  • Data package pour la gestion des stores, modèles et proxy.
  • Support complet pour MVC et MVVM.
  • Layouts puissants pour structurer l’interface utilisateur.
  • Outils de build et de packaging avec Sencha Cmd.

Exemple d'utilisation

Ext.application({
    name: 'MonApp',
    launch: function() {
        Ext.create('Ext.container.Viewport', {
            layout: 'fit',
            items: [
                {
                    xtype: 'panel',
                    title: 'Bonjour Ext JS',
                    html : '<h1>Bienvenue dans Ext JS</h1>'
                }
            ]
        });
    }
});

Avantages

  • Composants riches et cohérents pour des applications d’entreprise.
  • Gestion avancée des données et des stores intégrée.
  • Framework robuste pour des projets complexes et à grande échelle.
  • Documentation et support fournis par Sencha.
